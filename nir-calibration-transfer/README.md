# NIR Calibration Transfer Teaching

Corn NIR calibration transfer teaching page for Python Jupyter, Google Colab, and Orange Data Mining.

Live page: https://tai-shengyeh.github.io/nir-calibration-transfer/

Dataset: Eigenvector Research, NIR of Corn Samples for Standardization Benchmarking.

Raw MATLAB data is not stored in this GitHub Pages folder; the Python notebook/script downloads it automatically from Eigenvector Research. The Orange-ready CSV is in `orange/corn_nir_for_orange.csv`.
